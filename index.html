Projects

This is a collection of projects that I have made, all of them are web-based and use javascript and javascript based frame-works.
Click on any of them to check them out!

  • Javascript Calculator
    • This project is a simple calculator that is fully functional, and built entirely from scratch using a few javascript libraries: jQuery and a javascript keyboard shortcut library. The keyboard shortcut library was used to allow the user to use the keyboard as input.
  • Hiragana Writing Practice
    • This is a tool that I built for myself when I was learning Japanese in school. It randomly generates a sequence of hiragana character pronounciations for the user to attempt to translate, and then the user can check their translation against the correct one. This helped me learn the hiragana as well as practice my Japanese handwriting.
  • Wikipedia Viewer
    • This is basically a wikipedia search engine, I used the wikipedia api to allow the user to search wikipedia and find the article that they are interested in, there is also a random article button, which redirects the user to a random wikipedia article.
  • Tic Tac Toe
    • This is a web based tic tac toe game, the user can choose whether they want to play as an "x" or an "O", and they play tic tac toe against the computer. For now the computer just randomly pics a square, it doesn't see where the user placed his tile, I plan on adding this functionality soon.
